## Installation

- Check outthe last version on : [releases](https://github.com/andreiseverin/teylo-aim-training-mod/releases/) 
- Download the RAR file you want
- Copy all the files in your valve or ag folder depending on the mod you downloaded
- Lauch the game and choose "Create Game" 
- For Half-Life write `/train` in chat to display the menu
- For Adrenaline Game or Half-Life ag mini, make sure to change the gamemode to `train` first

## Features

- Classic Training

  	- Start/Reset training : Start a timer, give you infinite health, weapons and ammo and counts your kills
	- Start/Reset training - no ammo : Start a timer, give you infinite health and counts your kills
	- Stop training : (start training needed) Stop the timer and display statistics: kills/min
	- Add more bots : Open a menu to let you add a moving bot by level
	- Kick a bot: kick a random bot
	- Boost bots HP - 200/200 : Set the bots HP to 200/200 when they respawn
	- Freeze bots : Freeze all bots
	- Unfreeze bots : Unfreeze all bots
	- Killing Blue Fade <ON/OFF> : Set a blue fade effect on screen when doing a kill
	- Drop weaponbox <ON/OFF : Option for clearing weaponboxes droped by bots 
	- WallHack <ON/OFF> : Option to activate Wall Hack to see bots through walls

- Spawn Training
 	- Add a moving bot : Open a menu to let you add a moving bot by level
	- Add a stationary : Add a not moving bot 
	- Remove bots : Remove all bots
	- Show Spawn Info <ON/OFF> : Option to display in chat the location number on spawn
	- Show Damage info <ON/OFF> : Show the damage you dealt to the bots
	- Killing Blue Fade <ON/OFF> : Set a blue fade effect on screen when doing a kill
	- Drop weaponbox <ON/OFF : Option for clearing weaponboxes droped by bots 
	- WallHack <ON/OFF> : Option to activate Wall Hack to see bots through walls
